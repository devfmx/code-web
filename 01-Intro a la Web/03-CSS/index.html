CSS RESET

Cada vez que se inicia un proyecto con CSS es ideal que nos aseguremos que los estilos, sean como la idea original en cada uno de los exploradores que existen en el marcado.

Por eso a travez del tiempo muchos programadores han optado por la solucion de Eric Mayer CSS Rest

Nosotros usaremos y explicaremos el reset de JOHS COMEAU el cúal nos ofrece una solución mas moderna y que nos permite tener un mejor control de los estilos
